# 🏠 Real Estate Analysis Notebooks
한국 부동산 실거래·전세·가격지수 데이터를 분석하는 **Jupyter Notebook 컬렉션**입니다.  
지역별 매매 추세, 전세가율, 갭투자 위험도, 상승률 분석 등 다양한 분석 노트북을 정리합니다.

---

## 📌 소개

이 저장소는 한국 아파트 시장을 **데이터 기반으로 해석**하기 위한 분석 노트북 모음입니다.  
부동산 임장왕이 직접 수행한 다양한 분석을 Notebook 형태로 기록해두며,  
뉴스·콘텐츠 제작·실제 시장 이해에 활용 가능한 인사이트를 제공합니다.

---

## 📊 포함된 분석 종류

### ✔ 실거래 기반 분석
- 월별 매매가 추세 분석  
- 평형대별 가격 비교  
- 지역별 평균 매매가 변화  
- 아파트 단지별 가격 흐름  

### ✔ 전세·전세가율 분석
- 전세가율(매매 대비 전세 비율) 변화  
- 갭투자 위험도 분석  
- 지역별 전세가격 흐름  

### ✔ 가격지수 활용 분석
- 한국부동산원 매매/전세 지수 트렌드  
- 지역별 시장 온도차 분석  
- 상승/하락률 TOP 및 BOT 지역 찾기  

### ✔ 기타 확장 분석
- 전월세전환율  
- 지역 노후도(건축연도) 분석  
- 재건축 가능성 점수화  
- 시계열 기반 예측(ARIMA/LSTM 등)
